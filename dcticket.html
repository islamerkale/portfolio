Project Overview

DC Ticket is a mobile application designed to simplify the process of finding, contesting, and paying parking tickets in Washington DC. The app provides a user-friendly interface that helps citizens manage their parking violations efficiently.

User Research

Research Methodology

To understand the pain points and needs of users dealing with parking tickets, I conducted 15 user interviews with DC residents, surveyed 100+ respondents, and analyzed existing ticket payment solutions.

User Persona

Meet Sarah Johnson

Sarah represents our target user - busy professionals who value their time and want a straightforward way to handle parking violations.

User Journey Mapping

Current vs. Improved Journey

The user journey map compares the frustrating current experience with our proposed solution, highlighting key pain points and improvement opportunities.

Information Architecture

App Structure

The information architecture was designed to prioritize the most common user tasks while making all features accessible within 2-3 taps.

Wireframes

Low-Fidelity Exploration

The wireframing phase focused on establishing the core layout and interaction patterns before committing to visual design elements.

Style Guide & UI Kit

Design System

The UI style guide was created to ensure consistency across all screens and future iterations, including color palette, typography, and reusable components.

Prototyping

Interactive Prototypes

Using Figma, we created interactive prototypes to test key user flows before investing in high-fidelity designs.

Usability Testing

Testing Methodology

We conducted moderated usability tests with 12 participants representing our target demographics to identify areas for improvement.

Design Refinements

Iterations Based on Feedback

Based on usability testing results, we made several key improvements including enhanced visibility of contest options, simplified evidence upload, and improved payment confirmation.

High-Fidelity Mockups

Final Visual Design

The high-fidelity mockups incorporated the DC government's visual identity while maintaining a user-friendly, modern aesthetic.
